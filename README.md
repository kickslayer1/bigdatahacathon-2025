##  Project Overview

**Rwanda Trade Intelligence Dashboard** is a comprehensive web-based analytics platform developed for the NISR Data Science Hackathon 2025. This application transforms Rwanda's trade data into an interactive, government-ready policy instrument that supports evidence-based decision making.

###  Project Mission
To create a data-driven platform that identifies Rwanda's next biggest export opportunities and provides strategic insights for economic transformation, directly supporting the National Strategy for Transformation (NST2) 2024-2029.

##  Application Features

### **Core Analytics Components:**

1. **Interactive Dashboard**
   - Real-time trade performance visualizations
   - Export/Import trend analysis
   - Economic indicator tracking

2. **Market Intelligence Hub**
   - Rwanda's key export markets analysis
   - Trade partner performance metrics
   - Interactive data exploration tools
   - Chart.js powered visualizations

3. **Demand Prediction Engine**
   - Prophet-based forecasting models
   - 2026 market demand predictions
   - Commodity timeline analysis
   - Interactive prediction charts

4. **Policy Framework**
   - NST2-aligned strategic recommendations
   - Data-driven policy insights
   - Economic transformation roadmap

5. **Youth & SME Engagement Strategy**
   - Job creation analytics (1.25M target)
   - Skills development recommendations
   - Entrepreneurship opportunities

### **Technical Stack:**

- **Backend:** Python Flask with MySQL (JawsDB)
- **Frontend:** HTML5, CSS3, JavaScript
- **Data Visualization:** Chart.js, interactive dashboards
- **Machine Learning:** Prophet forecasting, Scikit-learn
- **Database:** Cloud MySQL with comprehensive trade datasets
- **Deployment:** Heroku with continuous integration
- **Authentication:** Secure user management system


##  Data Sources & Validation

- **Official Sources:** U.S. International Trade Administration
- **Government Data:** Rwanda National Strategy for Transformation (NST2)
- **National Institute of Statistics:** NISR official trade statistics
- **Rwanda National Bank:**BNR official trade statistics
